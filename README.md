## Quiz App
- Players: 2 (alternating or side-by-side)
- Resumable: Yes
- Config: Static questions in JavaScript (no external JSON)
 
# Requirements:
- 10 static multiple-choice questions.
- Players take turns (or both answer sequentially).
- Each question has 10-second countdown shown clearly in UI.
- Auto-submit on timeout.
- Track and display scores per player.
- Save/resume progress (question number, player scores, time left).

#  Stretch:
- Highlight correct answer.
- Allow users to select category and difficulty before starting the quiz.
